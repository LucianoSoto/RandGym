# RandGym

UI Angular sencilla para generar ejercicios aleatorios por grupo muscular.

## Requisitos
- Node.js 18+ (recomendado)
- npm

## Instalacion
```bash
npm install
```

## Ejecutar en desarrollo
```bash
npm run start
```
Luego abre `http://localhost:4200`.

## Ejecutar con VS Code (Run and Debug)
Se incluye una configuracion de depuracion para Angular:
- Abre el panel Run and Debug
- Selecciona "Angular: Serve + Chrome"
- Inicia la depuracion

Si VS Code solicita `launchSettings.json`, es porque esta intentando usar un perfil de .NET. Selecciona la configuracion Angular y asegurate de no usar un perfil de .NET.
